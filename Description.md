The Automated Leave Management System (ALMS) is a Java-based desktop application designed to streamline the leave request and approval process for faculty members in educational institutions.
The system provides a user-friendly Java Swing GUI with a modern gradient interface, integrated with a MySQL database for secure data storage and management.

The application supports user registration, login, leave application submission, leave tracking, and administrative approval/rejection functionalities.
An additional email notification feature is implemented using JavaMail API to keep faculty members informed about important account activities and leave status updates.
