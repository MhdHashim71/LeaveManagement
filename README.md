# Automated Leave Management System (ALMS)

The Automated Leave Management System (ALMS) is a Java-based desktop application designed to streamline the leave request and approval process for faculty members in educational institutions.

## Features

- User registration and login
- Leave application submission and tracking
- Administrative leave approval or rejection
- Email notification for leave status updates

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MhdHashim71/LeaveManagement.git
   ```
2. Open the project in your preferred Java IDE.
3. Build the project and resolve dependencies.
4. Run the application.

## Usage

1. Register as a new user or log in with existing credentials.
2. Submit a leave request through the application form.
3. Track the status of your leave requests.
4. Administrators can approve or reject leave applications.
5. Email notifications are sent upon status updates.
